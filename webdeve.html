  1. BOLD
      This is a physical tag,
      which is used to bold the text written between it.
  2. Strong
      This is a logical tag,
      which tells the browser that the text is important.
  3. Italic
      This is a physical tag which is used to make text italic.
  4. Emphasize
      This is a logical tag which is used to display content in italic.
  5. Small
      This tag is used to decrease the font size
      by one unit from base font size.
  6. Deleted
      This tag is used to display the deleted content.
  7. Insert
      This tag displays the content which is added
  8. BIG
      This tag is used to increase the font size by one conventional unit.
  9. SUP
      It displays the content slightly above the normal line.
  10. SUB
      It displays the content slightly below the normal line.
  11. UNDERLINE
      This tag is used to underline text written between it.
  1. HTML

    Primary Uses Key features
    Structuring web pages With tags and elements, we can define the headings, paragraphs, and other contents of a web page.
    Browsers come with a built-in stylesheet that visually differentiates these elements.
    Navigating the internet Navigating the internet would be much harder without HTML.
    Imagine having to manually enter the URL of every web page you wanted to visit.
    This would be our reality without anchor tags.
    Embedding images and videos HTML also allows you to not only embed images into a webpage but also adjust their width, height, position, and even the way they're rendered.


  2. CSS

    Primary Uses Key features
    Pages load faster Opportunity in Web designing:
    If anyone wants to begin a career in web designing professionally,
    it is essential to have knowledge of CSS and HTML.
    Solves a big problem Website Design: With the use of CSS, we can control various styles,
    such as the text color, the font style, the spacing among paragraphs etc...
    Provide more attributes Web Control: CSS has controlling power on the documents of HTML, so it is easy to learn.
    It is integrated with the HTML and the XHTML markup languages.


  3. JAVA SCRIPT

    Primary uses Key Features
    Scripting Language JavaScript is a lightweight scripting language made for client-side execution on the browser.
    Interpreter Based JavaScript is an interpreted language instead of a compiled one.
    In that sense, it is closer to languages like Ruby and Python. The browser interprets JavaScript's source code, line by line and runs it.
    Event Handling An event is an action or an occurrence in a system that communicates about said occurrence so that you can respond to it somehow.
